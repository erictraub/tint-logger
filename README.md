# **Tint**
Colors and Styling for your console.logs in the browser.


<br/>

##### Include:

```javascript
<script src="/...path-to.../tint-logger/dist/tint.js"></script>
```

or

```javascript
<script src="/...path-to.../tint-logger/dist/tint.min.js"></script>
```
